# lamp-installer-ubuntu
Bash Script to install Apache, PHP, MYSQL on Ubuntu Servef
LAMP is a combination of operating system and open-source software stack. The acronym LAMP came from the first letters of Linux, Apache HTTP Server, MySQL or MariaDB database, and PHP/Perl/Python.
THis Script install LAMP stack by running a single shell script on Ubuntu LAMP (Linux, Apache, MySQL and PHP, PhpMyAdmin) stack.

This Script works on all Ubuntu versions.

What this script does:
Install Apache Web server
Install PHP
Install MySQL
Install PhpMyAdmin

To run this script just type the following command on your terminal:

wget --no-cache -O - https://raw.githubusercontent.com/noibilism/lamp-installer-ubuntu/main/lamp-bash.sh | bash
